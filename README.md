# FIND YOUR MARVEL TWIN HERO
### 1. 클라이언트 화면 보기
- __heromatch > dist > index.html__

### 2. 클라이언트 화면 소스 파일 보기
- __heromatch > src__

### 3. 프로그램 (프로그램 실행 코드)
- __heromatch > programFile__

### 4. 서버 실행해보기
- 실행환경
> - node.js 설치
> - vue.js 설치
> - flask 설치
> - 파이썬 설치

- __모든 실행 전에 __
```
cd heromatch 
```

- 파이썬 라이브러리 설치 
```
pip install -r requirements.txt # 필요 라이브러리 한번에 설치
```

- 모든 실행 조건이 갖춰진 후
```
npm run dev
```
